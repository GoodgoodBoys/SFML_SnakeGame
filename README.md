# SFML_SnakeGame

This project is a practice project for the SFML library.

This project is developed and run using the Visual Studio IDE. The development environment is as follows:
* Windows Platform
* Visual Studio
* gcc version 11.2.0 (GCC)
* SFML 3.0.0

------------
